# DeepThought Daily Reflection Tree — Submission Pack
This project implements a deterministic reflection framework to analyze behavioral patterns across control, contribution, and perspective dimensions.

It includes:
- Structured decision tree logic (TSV)
- Visual flow diagram
- Design documentation
- Optional interactive agent
## Files
- `reflection-tree.tsv` — Part A tree as deterministic structured data
- `tree-diagram.md` — Mermaid diagram of the conversation tree
- `writeup.md` — Explanation of design choices, sources, and improvements
- `agent.py` — Optional Part B runnable CLI agent
- `sample-transcript-victim.md` — Optional transcript, victim/entitled/self-centric path
- `sample-transcript-victor.md` — Optional transcript, victor/contributing/altrocentric path

## Run Part B
```bash
python agent.py
```

Make sure `reflection-tree.tsv` is in the same folder as `agent.py`.
